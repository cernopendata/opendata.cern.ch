How to use the event display

The CMS event display (called iSpy Online) can be used to visualise in your browser collision data recorded by the CMS detector. Files of a custom type called .ig can be opened using iSpy Online. Each .ig file is in fact a collection containing numerous individual collision events.

This guide will take you through some of the basic functionality of the event display.
